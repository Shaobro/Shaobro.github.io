JavaScript数组API全集

Posted on 2021-01-14

前言

  JavaScript提供了丰富且强大的数组API，这使得前端开发更为便捷。

修改原数组的API

  方法：pop()
  描述：弹出数组最后一个元素,并返回这个元素。

  方法：push()
  描述：添加一个或多个元素到数组末尾，并返回新的长度。

  方法：shift()
  描述：弹出数组的第一个元素，并返回这个元素。

  方法：unshift()
  描述：在数组开头添加一个或多个元素，并返回新的长度。

  方法：splice(x,n,…)
  描述：从x位置开始截取，截取n个并返回被截取的元素，第三个参数是截取完了需要添加哪些元素，从x的位置开始添加。

  方法：reverse()
  描述：逆转数组的顺序。

  方法：sort()
  描述：给数组排序(默认从小到大)。

sort还可以接收一个函数作为参数，该函数接收两个参数 a, b
如果 a < b就返回一个负数，a就排在b的前面
如果 a === b 就返回0，顺序不变
如果 a > b就返回一个正数，a就排在b的后面

返回新数组的API

  方法：concat()
  描述：把传入的数组，或者元素+数组合并成一个新的数组，返回这个新数组。

  方法：slice()
  描述：截取数组，不会改变原数组。

从数组的第0位截取到数组的第3位

如果不给第二个参数，那么就从第一个参数的位置截取到数组的最后

  方法：join()
  描述：把数组中的元素连接成一个字符串并返回，不会改变原数组。

可以选择用什么分隔符，空字符就代表没有分隔符。

  方法：indexOf()
  描述：查找元素在数组中第一次出现的索引,如果没有就返回-1。

  方法：lastIndexOf()
  描述：查找元素在数组中最后一次出现的索引，如果没有就返回-1。

数组遍历方法

  方法：forEach()
  描述：传入一个函数，把数组的每一项作为函数的参数被调用，不改变原数组。

  方法：map()
  描述：传入一个函数，把数组的每一项作为函数的参数被调用，然后用函数每次的返回值组成新的数组并返回，不改变原数组。

  方法：filter()
  描述：传入一个函数，把数组的每一项作为函数的参数被调用，在函数内部判断符合条件的每一项，把它们组成新的数组并返回，不改变原数组。

  方法：reduce()
  描述：接受两个参数，函数和初始值，其中函数接收两个参数，那个初始值和数组的每一项，每次用初始值对数组的每一项进行操作并返回这个新的初始值(函数返回)，不改变原数组。

初始值为0，sum代表了初始值，每次用sum += item,最后初始值为21并被返回(reduce返回)

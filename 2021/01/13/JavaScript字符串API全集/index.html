JavaScript字符串API全集

Posted on 2021-01-13 Edited on 2021-01-22

前言

  一位前端开发工程师能走多远,原生JS的基础是必不可少的技能之一。趁着空闲时间把字符串和数组的API归下总。

JavaScript 字符串

  方法：charAt( index )
  描述：返回指定索引位置的字符。

  方法：charCodeAt( index )
  描述：返回指定索引位置字符的Unicode值。

  方法：concat( strX, strX, …, strX )
  描述：连接两个或多个字符串，返回连接后的字符串

  方法：fromCharCode( numX, numX, …, numX )
  描述：将Unicode转换为字符串。

  方法：indexOf( searchvalue, start )
  描述：返回字符串中检索指定字符第一次出现的位置。

  方法：lastIndexOf( searchvalue, start )
  描述：返回字符串中检索指定字符最后一次出现的位置;

  方法：localeCompare( target )
  描述：用本地特定的顺序来进行比较两个字符串。

配合排序调用：

  方法：match( searchvalue || regexp )
  描述：找到一个或多个正则表达式的匹配。

  方法：replace( regexp/substr, replacement )
  描述：替换与正则表达式匹配的字符串。

  方法：search( searchvalue )
  描述：检索与正则表达式相匹配的值，如果没有找到任何匹配的子串，则返回-1。
  注意：search()方法对大小写敏感。

  方法：slice( start, end)
  描述：提取字符串的片段，并在新的字符串中返回被提取的部分。

  方法：split( separator, howmany )
  描述：把字符串分割为子字符串数组。

  方法：substr( start, length )
  描述：从起始索引号提取字符串中指定数目的字符。

  方法：substring( start, stop )
  描述：提取字符串中两个指定的索引号之间的字符。

  方法：toLocaleLowerCase()
  描述：根据本地主机的语言环境把字符串转换为小写。

  方法：toLocaleUpperCase()
  描述：根据本地主机的语言环境把字符串转换为大写。

  方法：toLowerCase()
  描述：把字符串转换为小写。

  方法：toUpperCase()
  描述：把字符串转换为大写。

  方法：toString( radix )
  描述：返回字符串对象值。

  方法：trim( radix )
  描述：移出字符串首尾空白。

  方法：valueOf()
  描述：返回某个字符串对象的原始值。
